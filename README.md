### Snake
SnakeGame.java contains code for the classic game Snake. In addtion to the seed prompt version, I added prevention of direction reversals caused by clicking two arrow keys within the same movement frame and improved the snake UI by adding a head and tail. On the main game page, I added a title and a start button. I also added the high score feature, which keeps track of the highest score across all games during which the program was running and also lets the user know when they broke the high score on the Game Over page. In summary, this game lets players use the arrow keys to change the direction of a moving snake that gets bigger every time it eats food. Guide the snake to food, don't crash into walls or the body of the snake, and try to beat your high score!

I learned that agentic AI works better when you provide details, including specifics about what you want done and how you want it done. I also learned that it is super important to understand, review, and test the code agentic AI generates. This ensures that you understand how your program works, which enables you to make changes more confidently in the future. It also ensures that app behavior is known through testing and edge cases are found.
